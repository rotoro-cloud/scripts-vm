Чтобы передать несколько переменных в awk, нам нужно добавить флаг -v 
перед каждым синтаксисом назначения переменных. 
Переменные расширяются внутри блока действий без каких-либо специальных символов. 

У нас здесь заготовка скрипта `parking-checker.sh`, который анализирует файл `parking.txt`, отражающий парковочные места на многоэтажной парковке. 
Столбцы под буквами `a-h` обозначают машино-место, а строки являются этажом здания парковки.

В этом файле `X` означает, что место занято, а `O` означает, что автомобиль может быть припаркован.

Целью скрипта `parking-checker.sh` является ответ, свободно ли определенное место или нет. 

Есть пара особенностей этого скрипта. 
Во-первых данные для работы он получает из переменных окружения оболочки: $LITER и $FLOOR, которые обозначают место и этаж соответственно.
Второй момент в том, что для человеческой наглядности первая строка и первый столбец являются информационными и не участвуют в работе.
(Ознакомься с файлом parking.txt, чтобы понять о чём я)

```
cat parking.txt 

* a b c d e f g h
1 X O O O O X O X
2 X O X X O X X X
3 O O O O O O O O
4 X X X X O X X X
5 X O O O O X O O
6 O O O O O O O O
```

Таким образом, переданные в скрипт этаж и место будут иметь значения на один больше в файле. 
Т.е. если мы хотим проверить второй этаж и место `c`, то это будет третья строка и четвёртый столбец.

Пример запуска в данный момент:

```
export LITER=3
export FLOOR=2
./parking-checker.sh parking.txt

Checking for floor #2 and liter 3
```

Поскольку мы передали в скрипт `3-й` столбец, `2-ю` строку, мы должный получить пересечение столбца под литерой c и строки под номером 2. 
Строка с номером 2 фактически является третьей линией файла, а столбец c - четвёртым столбцом. Это будет `X`.


_ЗАДАНИЕ:_

Измени файл `parking-checker.sh`, добавь внизу команду `awk`, которая получает переменные скрипта `${column}` и `${row}` и печатает значение, содержащееся в этой позиции. 
Путь к файлу скрипт должен получать в виде аргумента командной строки. Не забудь сделать этот скрипт исполняемым.
